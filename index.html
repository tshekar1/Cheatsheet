HTML Tables

Tag Description
<table> Encloses all of the table contents.
<thead> Name used for the part of the table (the first row) that contains the column headings. Content of each box in the row should be entered as a list wihthin this element enlcosed in <th>
<th> Wrapped around the content of each box in the heading row, typed as a list within <thead>, also used for row headings
<tbody> The rest of the rows, i.e. the body of the table should be enclosed within this.
<tr> Eleement correpsonding to each row of the table. Contents of each box in a row should be enlosed within <td> (table data) elements as a list.
<td> Wrapped around the content of each box in a row, typed as a list within <tr>
<tfoot> Used for the footer section of the table.
